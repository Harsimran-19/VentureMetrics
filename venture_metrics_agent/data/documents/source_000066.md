# 研究所及研究中心

中大的學術部門積極參與廣泛的研究領域。大學設有多個研究所及研究中心，於優質的跨學科研究中各有專精。此外，不少院系均設有較小型的研究服務單位。大學亦與多個內地及海外學術機構及院校合作，成立聯合研究單位，以結合專才和資源，促進不同學科領域的研究。

研究所及研究中心的成功將根據各種指標進行評估，包括研究活動、收到的研究資助、收到的捐贈、研究論文以及其產生的影響。
