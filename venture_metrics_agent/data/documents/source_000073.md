# 樂施毅行者2026 – 每年不變的約定

由樂施會主辦、一年一度全港最大型遠足籌款活動「樂施毅行者」，今年將於11月20至22日舉行，並以「每年不變的約定— 樂施同行50年，為改變而行」為主題，除了經典的 100 公里麥理浩徑挑戰，今年更特別增設「樂施會50周年限定版」20公里路線，讓更多人加入「為改變而行」的行列。樂施會今年特意邀請黃浩然、坤哥、蔡寶欣、曾令令一同現身「樂施毅行者2026」全新宣傳影片，分享毅行精神！
