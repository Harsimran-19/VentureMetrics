# SMART Marketplace

An online platform that provides easy access to SET and TFEX data via API

[SET Information Services Overview](/en/services/connectivity-and-data/data/main)[Real time / Delay](/en/services/connectivity-and-data/data/realtime)[End of day / Historical](/en/services/connectivity-and-data/data/end-of-day)[Data Request](https://www.set.or.th/en/services/connectivity-and-data/data/data-request)[SETSMART](/en/services/connectivity-and-data/data/web-based)[SMART Marketplace](/en/services/connectivity-and-data/data/smart-marketplace)

Data on SMART Marketplace via API

### Historical Intraday Trading Data

* Equity & Derivatives Trading Ticker
* Equity & Derivatives Bids/Offers
* SET Index series data

### Intraday Trading Data (Snapshot)

* Equity Real-time/Delayed Trading data
* Equity Real-time/Delayed Bids/Offers
* Real-time/Delayed SET Index series data

### Equity EOD Data (OAQ)

* Equity EOD trading price and statistics
* EOD SET index series data
* Market trading categorized by investor types

### Derivatives EOD Data (ODQ)

* Derivatives EOD trading data
* Market trading categorized by investor types

### One Report

Listed Company Data Data which report annually by SET's listed companies in the form of structured data including Business Operations and Performance, Corporate Governance, and Financial Statement

[Try this service free of charge in 2024](https://www.set.or.th/app/online-data/one-report?lang=en)

### ESG Data

ESG data of listed companies in the Stock Exchange of Thailand in the form of structured data including Environmental, Social and Governance information

[Try this service free of charge in 2024](https://www.set.or.th/app/online-data/esg?lang=en)

### Reference Data and Corporate action

* Profile of securities to be traded on the next trading day
* Corporate action data of listed companies

### Company Fundamental Data

* Equity Historical EOD trading price and statistics
* Financial data and ratio

[Try this service free of charge for SETSMART Members](https://www.set.or.th/app/online-data/fundamental-data?lang=en)
