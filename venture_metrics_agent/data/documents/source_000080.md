# 创启未来

![Image 8](https://www.pkusp.com/uploadfile/202512/56a08b654e5df75.png)

 “创启未来”国际青年科技创业大赛启动于2015年，以“支持创新、服务创业”为宗旨，聚焦前沿科技领域，关注热点创新主题，整合双创服务资源，深度挖掘人才和项目，为战略新兴产业发展寻找最先进的技术解决方案，为创业者提供项目展示平台、交流平台和持续成长空间。

# 北大孵化器

![Image 39](https://www.pkusp.com/uploadfile/202512/290d996fe1ad02b.png)

北大孵化器成立于2002年，是国家级科技企业孵化器、国家级人工智能专业化众创空间，为高校科研成果转化、科技企业孵化、创新创业人才培养提供专业科技服务。服务体系包含物业服务，工商财税法品牌咨询等公共服务，政策收集整理发布申报等政务服务，投融资服务、转板上市等金融服务，对接市场订单等市场服务，产业落地引导等产业服务，与企业从0到1共同成长。

# 先锋加速器

![Image 49](https://www.pkusp.com/uploadfile/202512/cb5adfcbac7cd03.png)

先锋·加速器是北大科技园为助力全国创新企业加速成长而打造的专业科技服务赋能平台，致力于帮助具备先锋创新精神的科技型企业快速成长，通过先锋加速计划为企业提供技术、资本、产业、市场多方位服务，跨越企业生命周期关键阶段。

##### 先锋·加速器 一期计划

合作伙伴：北京智源人工智能研究院、百度智能云

聚焦领域：人工智能、物联网
