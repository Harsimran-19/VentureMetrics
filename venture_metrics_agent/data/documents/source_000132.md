# **成电资源**

电子科技大学坐落于四川省成都市，1956年创建而成，学校1960年被中共中央列为全国重点高等学校，1997年确定为国家首批“**211工程**”建设的重点大学，2001年进入国家“**985工程**”重点建设大学行列。现如今，电子科大正在大力实施三大行动，努力在“十三五”期间建成电子信息领域世界一流、理工深度融合的研究性大学，尽早建设成为“中国特色、世界一流”的双一流大学。当前，电子科技大学已有国家一级重点学科**2**个，国家级重点实验室**5**个，国家级特色专业建设点**14**个，在国内电子信息学科领域处于领先地位，其中电子科学与技术全国排名第一；信息与通信工程全国排名并列第一。

# **项目介绍**

电子科技大学国家大学科技园（天府园）是推动电子科技大学科技成果转化、创业企业孵化、创新创业人才培养的科技创新平台，是电子信息产业发展的“国家队”，是成都市校院企地合作的先行者和开拓者。成都科杏投资发展有限公司是科技园的唯一建设运营方，下设的全资子公司明思成都商务服务有限公司主要负责科技园的物业服务和智慧园区平台运维。

园区位于天府新区双流辖区，总占地面积约 460亩，规划建筑面积 60万 平方米，总投资约 40亿 元，主要布局集成电路、大数据与人工智能、通信与物联网等高新技术产业，构建从苗阜、孵化、加速器、规模化的完整的科技产业生态链，已经吸引了20 0 余家电子信息高新技术企业入驻。未来，园区最终将聚集超**10 00家**科技型企业，实现超**100亿**产值、超**8亿**税收、超**2.5万**科技精英人才的就业，打造中国最好的电子信息科技园。
