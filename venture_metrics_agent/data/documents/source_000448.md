# 2023 ABS - Characteristics of Business Owners (Employer Businesses)

 April 2023 

[](https://www.census.gov/data/tables/2023/econ/abs/2023-abs-characteristics-of-owners.html)

The Annual Business Survey uses the collection year in the survey name rather than a single reference year.This is to better reflect the collection cycle of the survey. Therefore, the 2023 ABS covers reference year 2022.

Multiple owner characteristics are included in the table listed above. Below are the links to the data table by owner characteristics topic with Question Description codes for the topics are in parenthesis.

For more information on using the Census Bureau’s data platform, visit the data.census.gov[resources page](https://www.census.gov/data/what-is-data-census-gov.html).
