# 北京市科学技术委员会、中关村科技园区管理委员会2025年政府信息公开工作年度报告

发布时间： 2026-01-19 来源：市科委、中关村管委会

（一）加强组织领导

围绕全年目标任务，强化组织领导，对政府信息公开相关工作进行部署。健全委领导牵头、相关部门协同参与的工作体系，适时研究调度，动态跟踪进展，推动各项工作扎实落地、取得实效。

（二）主动公开情况

标准化规范化做好政府信息公开专栏建设，及时准确更新政府信息公开指南、年报、专栏内容。持续加强重点领域信息公开，围绕国际科技创新中心建设，及时发布三城一区、深化改革、高精尖产业、园区风采、开放合作、科技前沿等最新动态信息。加强政策文件发布与解读，充分运用文字解读、一图读懂、数字人等丰富解读形式。深化公众参与，做好政策性文件发布前向社会公开征集意见工作。强化政务公开全清单管理，推动主动公开依法依规、规范有序。

（三）依申请公开情况

本年度新收政府信息公开申请43件，办理上年结转1件。新受理中，包括网页申请39件、信函申请2件、当面提交2件，除1件结转下年度办理外，其余均已按程序办结。妥善办理1起信息公开行政复议案件，维持答复。

（四）政府信息管理情况

严守政府信息公开安全底线，健全信息发布全周期管理机制。针对通过门户网站、政务新媒体等平台发布的政策文件、通知公告等政府信息，认真落实“先审查、后公开”；发布前，严格履行保密审核和信息公开审查程序，确保源头安全；发布后，实施常态化安全监测与定期风险排查，确保公开信息的持续安全可控。

（五）政府信息公开平台建设情况

推动市科委中关村管委会官网功能升级，官网主页改版为横版布局、核心信息一屏化呈现，创新上线首页自动播报与数字人交互功能，大幅提升信息获取便捷性。持续深化政务新媒体建设，强化平台协同与内容创新。深入对接“京策”平台，提高政策预制、推送、兑现等服务水平，提升数据管理和政策服务效能。

（六）教育培训与监督保障情况

及时掌握并认真落实政务公开和政务服务领域最新部署要求，积极参加全市政务公开工作培训。市科委中关村管委会举办的年度依法行政培训班，专题设置信息公开方面培训内容，进一步提升全委系统信息公开能力和水平。
