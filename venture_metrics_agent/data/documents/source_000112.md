# 工大飞天科技企业孵化器

发布时间：2024-07-05 点击数：620 

工大飞天科技企业孵化器是西北工业大学国家大学科技园“创业苗圃+众创空间+孵化器+加速器+产业园”全链条孵化载体的重要组成部分。打造了“物理空间+治理空间+产业空间”三位一体的创业生态体系。建设了智能无人系统公共技术服务平台，获批了秦创原立体联动孵化器特色载体示范项目。

2021年获批西安市科技企业孵化器，2022年获批陕西省科技企业孵化器。
