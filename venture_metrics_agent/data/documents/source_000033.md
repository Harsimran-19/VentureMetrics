# 深港澳科技联盟成立

2019年04月04日
来源：南方网

　　南方网讯  4月3日，深港澳科技联盟正式成立，今后将整合深港澳科技创新和产业资源，形成科技创新和产业新动力，为粤港澳大湾区建设提供创新和产业支撑。深圳市人大常委会副主任、市科协主席蒋宇扬出席会议。

　　深港澳科技联盟领导机构将由理事会承担，深港澳各设一位主任，首任主任分别是深圳科协党组成员、副巡视员孙楠、香港资讯科技联会会长邱达根、澳门科技大学副校长姜志宏。

　　“建设国际科技创新中心是粤港澳大湾区的重要使命，深港澳拥有数量众多的科技企业及科技工作者，联盟作为推动深港澳科技交流与合作的重要载体，已具有良好的组织和人脉优势。”蒋宇扬强调，深港澳要充分发挥联盟开放型、枢纽型、平台型的功能作用，动员政府、企业、社会各方力量，优化、整合各类科技创新和产业资源，围绕粤港澳科技领域的重点问题，开展战略性、前瞻性、系统性研究。

　　深圳市科学技术协会党组书记林祥表示，科协主要职能之一是开展民间科学技术交流活动，近年来，深圳科协与港澳科技社团等机构已建立了广泛联系，在促进学术交流、科技成果转化、科技交流活动等领域的合作渠道畅通。“深港澳科技联盟的成立，将为我们今后联系和服务提供新的重要平台，深港澳科技专家群体更可以因此而形成合力，为深港澳科技发展发力。”林祥说。

　　林祥介绍，该联盟重点将推动科技创新、产业法规和政策体系的建立、健全和完善，并探索建立新时期科技创新和产业的发展模式，以及人才的有效培养和使用模式，打造大湾区科技创新和产业生态圈。

　　除了整合资源外，联盟下一步将开展大湾区科技状况调查，掌握科技创新和产业发展状况，为政府决策提供参考依据。其次，借鉴、总结、推广各地科技创新和产业发展的成功做法和经验，在开展科技创新和产业发展理论研讨的同时，开展科技交流和培训。此外，搭建科技活动策划、科技创新和产业研发平台、联络交流平台，提升联盟成员单位综合实力。

　　深港澳科技联盟成立大会由深圳市科协主办，深圳市科技开发交流中心等承办，来自深港澳共100多名科技界代表参与会议。 （全媒体记者/张玮）
